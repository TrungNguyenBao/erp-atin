Interaction & Accessibility Notes

Trigger: "+ Create Sprint" button (top-right of Backlog header) OR dashed placeholder at bottom of backlog.

Duration chips: Selecting a chip auto-calculates End Date = Start Date + N weeks (business days). Selecting "Custom" enables manual End Date input.

Start Date default: Day after active sprint's end date. If no active sprint, defaults to today.

Name auto-generation: "Sprint N" where N = last sprint number + 1. Editable.

Capacity section: Collapsed by default. Expanding loads team members from project. SP estimate = (days × velocity_per_day). Read-only estimate, not stored.

Validation: Sprint Name + Start Date required. End Date required but auto-filled. Goal is optional.

Keyboard: Tab navigation through all fields. Enter submits form. Escape closes modal (with confirmation if dirty).

After create: Modal closes → success toast "Sprint 5 created" → new sprint row appears in backlog with blue highlight (fades after 3s) + Draft badge.

ARIA: role="dialog", aria-labelledby="modal-title", focus trapped inside modal, focus returns to trigger button on close.

Overlap check: If start/end dates overlap with an existing draft sprint, show inline warning (not a hard block).
