1. Project Dashboard (Overview)

Mục tiêu: Tổng quan tình trạng dự án

Features:
Sprint hiện tại (progress, burndown)
Tổng số task theo status
Velocity team
Deadline / milestone
Activity gần nhất

👉 Đây là màn hình executive view (PM/Manager)

📋 2. Product Backlog

Mục tiêu: Quản lý toàn bộ yêu cầu (User Story / Task)

Features:
List backlog items
Priority (High / Medium / Low)
Estimate (Story Point)
Filter / search / tag
Drag & drop reorder
Attachments / description / acceptance criteria

👉 Đây là trái tim của Agile

🏃 3. Sprint Board (Kanban Board)

Mục tiêu: Quản lý công việc trong sprint

Columns:
To Do
In Progress
Code Review
Testing
Done
Features:
Drag & drop task
Assign user
Due date
Quick edit

👉 Screen được dùng hàng ngày bởi dev team

📅 4. Sprint Planning Screen

Mục tiêu: Lên kế hoạch sprint

Features:
Chọn backlog items vào sprint
Capacity team (giờ / story point)
Estimate effort
Auto suggestion (AI nếu có)

👉 Quan trọng cho PM + Tech Lead

📊 5. Reports & Analytics

Mục tiêu: Tracking hiệu suất

Reports:
Burndown Chart
Burnup Chart
Velocity Chart
Cycle time
Lead time

👉 Dùng để data-driven decision (rất hợp với mindset của bạn)

👥 6. Team Management

Mục tiêu: Quản lý user & role

Features:
Role (Admin / PM / Dev / QA)
Permission
Workload view
Performance tracking
🧾 7. Task Detail Screen

Mục tiêu: Xử lý chi tiết từng task

Features:
Description
Checklist
Comments
Attach file
Activity log
Subtasks

👉 Đây là nơi collaboration diễn ra nhiều nhất

💬 8. Collaboration / Communication

Mục tiêu: Giao tiếp trong team

Features:
Comment thread theo task
Mention (@user)
Notification
Chat (optional)
🔔 9. Notification Center

Mục tiêu: Không miss thông tin

Features:
Task assigned
Status changed
Mention
Deadline warning
⚙️ 10. Project Settings

Mục tiêu: Cấu hình project

Features:
Workflow config (custom status)
Sprint duration
Estimation type (Story point / hours)
